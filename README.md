# India Terrorism Trend Analysis (Tableau vs Power BI)

## Project Objective
This project analyzes terrorism incidents in India over time to identify 
temporal and geographical trends.

The same set of five visualizations were recreated in both Tableau and 
Power BI to evaluate similarities, usability differences, and analytical capabilities.

---

## Dataset
Public terrorism dataset covering incidents in India.
Details available in Dataset folder.

---

## Key Visualisations
1. Severity by Attack Type
2. Top 15 states under Attack
3. Attack type by Target
4. Trend over Time
5. Geographical Hotspots
